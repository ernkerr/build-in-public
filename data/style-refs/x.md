### 2026-04-14 — "Deleted 13,000 lines"
```
Deleted 13,000 lines of my own code this week

It was a Next.js web app: Supabase, OAuth, a settings page, the works

Rebuilt the whole thing as a Claude Code skill w/ markdown + a few scripts

Same features, just lives inside my terminal now 👾

#buildinpublic
```

Why it works:
- Strong hook with a concrete number
- Sets up the contrast: heavy stack → lightweight primitive
- Ends with a relatable punchline + single emoji
- No periods at line ends = reads fast, feels casual
